Descripción de la instalación
=============================

Plone 4.3.x no es un software monolítico, sino que está construido de 
docenas de diferentes paquetes de software, escritos por cientos de 
programadores alrededor del mundo. Esta diversidad acarrea complejos 
problemas de administración de configuraciones. Estos problemas se 
solucionan usando una herramienta como :ref:`Buildout <que_es_buildout>`.

Requerimientos de hardware
==========================

La configuración mínima para poder ejecutar Plone son:

- Procesador: 2 procesadores.

- Memoria RAM: 2 GB.

- Disco duro: 10 GB + espacio ocupado por los datos.

- Conexión a internet de banda ancha.
